# AirWise Notion Operating System

Last updated: 2026-04-15

## Purpose

Run AirWise with a lightweight founder operating system that matches the company’s actual stage: one working product repo, a pilot-oriented build cycle, and a need to separate current product truth from future roadmap work.

## Recommended top-level areas

- Weekly priorities
- Product truth and docs
- Pilot buildings
- Filing and compliance reviews
- Monitoring/runtime risks
- GTM and pilot stakeholders

## Key operating rule

Any roadmap or planning page should clearly distinguish:

- implemented now
- in current sprint
- later roadmap

That rule now matters because the product has enough breadth that planning artifacts can drift ahead of the shipped system.
